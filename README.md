# Nascimento Advogados — Site institucional

Site institucional do escritório **Nascimento Advogados Associados**, com foco em direito previdenciário e trabalhista.

## Publicação

O site é estático (HTML, CSS e JavaScript em um único arquivo). Para publicar:

1. Envie os arquivos desta pasta para a hospedagem ou use GitHub Pages.
2. Defina `index.html` como página inicial.

## Estrutura principal

- `index.html` — página do site
- Imagens (`hero.jpg`, `dr-wellington.jpg`, logos, reels, etc.)
- `assets/` — cópias de apoio das imagens
